  <h1>Política de Privacidad</h1>
  <p>QA Evidence Reporter recopila únicamente la información necesaria para documentar incidencias durante pruebas de calidad (QA).</p>
  <p>La extensión puede recopilar:</p>
  <ul>
    <li>URL de la página donde ocurre el problema.</li>
    <li>Capturas de pantalla generadas por el usuario.</li>
    <li>Descripciones del resultado esperado y obtenido introducidas por el usuario.</li>
    <li>Información técnica relacionada con errores y solicitudes de red asociadas a la incidencia.</li>
  </ul>
  <p>La información recopilada se utiliza exclusivamente para generar y almacenar reportes de errores en Google Sheets y Google Drive.</p>
  <p>QA Evidence Reporter no vende, comparte ni utiliza la información recopilada con fines publicitarios o comerciales ajenos a la funcionalidad de la extensión.</p>
  <p>Toda la recopilación de datos se realiza únicamente cuando el usuario solicita explícitamente generar un reporte.</p>
